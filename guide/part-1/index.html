    File Structure

    A premade folder structure and starter files can be downloaded here. Feel free to rearrange things how you see fit.

    Software

    To get things started we'll need to make sure we have some software installed. These are just suggestions. If you use alternatives that you are more comfortable with then you should use those.

    • Sublime Text - Code editing
    • Blender3D - 3D modelling
    • Krita - Image editing
    • Python 2.7 - Local server

    Local Web Server

    THREEjs' asset loading functions require a local web server. Set one running by going to the command line, navigate to the workshop folder and type python SimpleHTTPServer 8000 Open your browser and enter http://localhost:8000 into the address bar.
